# Windows-Server-Active-Directory-Lab
## Overview

This project documents the creation and management of a Windows Active Directory home lab environment using Windows Server 2022 and Windows 11 virtual machines.

The lab was built to gain hands-on experience with:
  
## Skills Demonstrated
- Active Directory Administration
- Group Policy Management
- DNS Configuration
- Windows Server Administration
- Virtualization
- Troubleshooting

## Environment 

| Component      | Details                         |
| -------------- | ------------------------------- |
| Host Machine   | Dell OptiPlex 5050 Mini         |
| Host OS        | Windows 11 Pro                  |
| Virtualization | VMware Workstation  
| Server OS      | Windows Server 2022             |
| Client OS      | Windows 11                      |
| Domain Name    | lab.local                       |
